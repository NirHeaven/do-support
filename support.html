常见问题

如何添加新的待办事项？

1. 点击主界面右下角的"+"按钮

2. 输入待办事项内容

3. 设置优先级（可选）

4. 点击确认添加

如何标记任务完成？

1. 在任务列表中找到要完成的任务

2. 点击任务前的复选框

3. 任务会自动移至已完成列表

如何恢复已删除的任务？

目前版本暂不支持恢复已删除的任务，请谨慎删除操作。我们建议：

  • 删除前确认是否真的需要删除
  • 如果不确定，可以先将任务标记为完成

FAQ

How to add a new to-do item?

1. Click the "+" button in the bottom right corner

2. Enter the to-do item content

3. Set priority (optional)

4. Confirm to add

How to mark a task as completed?

1. Find the task in the task list

2. Click the checkbox in front of the task

3. The task will automatically move to the completed list

How to recover deleted tasks?

The current version does not support recovering deleted tasks. We recommend:

  • Confirm if you really need to delete before deleting
  • If unsure, mark the task as completed instead
